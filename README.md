# Restaurant-menu
🍽️ The Spice Route - Restaurant Menu UI/UX Design
📌 Project Overview:
The Spice Route is a modern restaurant menu UI/UX design created in Figma. The design focuses on providing a visually appealing and user-friendly experience, helping customers easily browse menu items and place orders.
✨ Features
1)Clean and modern interface
2)Attractive hero section with food imagery
3)Categorized menu sections (Veg, Non-Veg, Starters, Biryani, Desserts)
4)Easy-to-read dish cards with images and pricing
5)Prominent "Add" buttons for quick ordering
6)Responsive and user-friendly layout
7)Footer with contact details and social media links
🎨 Design Elements
Primary Color: #FF6B35
Background Color: #FFF8F0
Typography: Poppins
Tool Used: Figma
🎯 Objectives
1)Create a visually appealing restaurant menu
2)Improve menu navigation and usability
3)Organize dishes into clear categories
4)Enhance user experience through simple and clean design
